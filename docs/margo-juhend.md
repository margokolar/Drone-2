# Juhend Margole: kuidas äppi vaadata ja edasi arendada

See juhend on kirjutatud sulle, Margo. Siin pole midagi tehnilist — ainult sammud, mida
ise teed. Kõik keerulise teeb AI agent Cursoris.

---

## 1. Veebiversioon (töötab juba täna)

Äpi veebiversioon töötab edasi täpselt nagu seni:

1. Ava iPhone'is Safari ja mine äpi aadressile (sama, mida seni kasutanud oled).
2. Kui oled äpi koduekraanile lisanud ("Lisa avakuvale" / "Add to Home Screen"), siis
   ava see sealt.
3. Veebiversioon uueneb ise — kui agent on muudatuse valmis teinud ja üles laadinud,
   piisab äpi sulgemisest ja uuesti avamisest.

Veebiversioon jääb alles ka pärast seda, kui päris iPhone'i äpp valmis saab. See on hea
varuvariant, kui telefoni-äpiga peaks midagi juhtuma.

## 2. Päris iPhone'i äpp (kui see on telefoni pandud)

Kui esmane seadistus on tehtud (poeg aitab sellega ühe korra), on äpp sinu telefoni
koduekraanil nagu iga teine äpp. Ava ja kasuta.

### Esimesel korral: luba arendaja

Kui äpp esimest korda avamisel ütleb, et arendajat ei usaldata:

1. Ava telefonis **Seaded (Settings)**
2. **Üldine (General)** → **VPN ja seadmehaldus (VPN & Device Management)**
3. Vajuta oma Apple ID nimele ja siis **Usalda (Trust)**
4. Ava äpp uuesti

### Tähtis teada: äpp lakkab töötamast iga 7 päeva järel

Kuna me ei ole (veel) ostnud Apple'i arendajakontot, lubab Apple äpil telefonis töötada
korraga ainult 7 päeva. Pärast seda äpp lihtsalt ei avane enam — see on normaalne, midagi
pole katki ja sinu seaded/presetid ei kao kuhugi.

**Mida teha, kui äpp enam ei avane:**

1. Ühenda iPhone kaabliga oma Maci külge
2. Ava Macis Cursor (see sama programm, kus sa agendiga räägid)
3. Ütle agendile: **"Pane äpp uuesti mu telefoni"**
4. Oota, kuni agent ütleb, et valmis (paar minutit), ja proovi äppi telefonis avada

Sama moodi tee siis, kui tahad lihtsalt kõige värskemat versiooni telefoni saada.

## 3. Kuidas uusi asju küsida

Täpselt nagu seni:

1. Ava Cursor ja kirjelda agendile oma sõnadega, mida tahad — näiteks
   *"lisa metronoomile valjem klõps"* või *"burdooni nupp võiks olla suurem"*
2. Agent teeb muudatuse ära
3. Kui tahad seda kohe telefonis näha, ühenda telefon kaabliga ja ütle agendile:
   **"Pane äpp mu telefoni"**
4. Proovi järele ja ütle agendile, kas on hea või mida muuta

Kui midagi läheb katki või on imelik, kirjelda agendile võimalikult täpselt, mida sa
tegid ja mida nägid/kuulsid. Agent oskab ise vea üles otsida.

## 4. Reeglid, mis hoiavad elu lihtsana

- **Kaabel:** telefoni äpi uuendamiseks peab iPhone olema kaabliga Maci küljes.
- **Ära ava Xcode'i** (see suur Apple'i programm) — sul pole seda kunagi vaja, agent
  teeb kõik ise.
- **Kui äpp ei avane** — see on peaaegu kindlasti 7 päeva reegel. Kaabel külge, Cursor
  lahti, "pane äpp uuesti mu telefoni".
- **Veebiversioon on alati olemas** Safaris, kui telefoni-äpp parasjagu ei tööta.

## 5. Tulevikus (kui otsustame osta)

Apple'i arendajakonto maksab ~99 $ aastas. Kui see kunagi ostetakse:

- 7 päeva piirang kaob ära
- Uuendused hakkavad tulema **TestFlighti** äpi kaudu — vajutad telefonis lihtsalt
  "Update", kaablit pole enam vaja
- Saab äppi jagada ka sõpradele

Seni proovime tasuta variandiga — kui iganädalane kaabliga uuendamine ära tüütab, on see
märk, et arendajakonto on oma hinda väärt.
